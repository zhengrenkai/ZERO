## 进化节奏

### 每次对话自动检查一次

```
1. 还差什么技能没学会（路线图上还剩几项）
2. 上次学到的东西有没有用上（优化效果回看）
3. 系统健康度是涨了还是跌了
4. 如果有卡住的地方，提示少爷
```

### 进化规则

1. **一次只学一件事**。不要同时学复盘 + 情绪记录 + 独立执行卡
2. **先有模板，再填充内容**。结构到位了，内容自然来
3. **检验标准是"是否被用过一次"**。学会了但从未使用 = 白学
4. **中断不是失败**。学了一半放下，下次捡起来继续
