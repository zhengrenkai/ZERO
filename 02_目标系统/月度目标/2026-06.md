## 本月核心目标

### 🛌 睡眠（第一优先级）
- [ ] 每周至少 4 天在 **23:30 前躺下**
- [ ] 23:00 手机放客厅 ≥ 16 天
- [ ] 如果某天没做到，第二天不补、不追、不自责，直接重启

### 💪 身体线（维持）
- [ ] 维持最低训练版不断线
- [ ] 不主动进阶（引体不加、倒立不加）
- [ ] 允许缩水版：引体 2×7 + 猫牛式 = 过关
- [ ] 连续 streak 目标：月底冲到连续 30 天

### 💼 现金流稳定线（加速）
- [ ] 每周 4 次 25min 低门槛推进（刷题/简历/小专题）
- [ ] 优先安排在 **早起后第一件事**（精力峰值时段）
- [ ] 完成简历骨架恢复
- [ ] 完成 1 个算法小专题的重新启动
- [ ] 本月目标：从"偶尔破冰"到"每周稳定 4 次"

### 📈 交易线（降频保接触）
- [ ] 每周 2 次最低动作（看课 15min / 标记结构 / 语音笔记 三选一）
- [ ] 不要求产出，不要求日更
- [ ] 中断超过 3 天不补，直接重启
