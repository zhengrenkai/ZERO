# 财务自由路径

> 目标不是把自己锁死在某一种赚钱方式里，而是逐步降低对单一工资的依赖。

---

## 一、最终目标

长期方向：

```
身体稳定 → 精力稳定 → 工资现金流提高 → 交易/副业能力形成 → 稳健资产打底 → 自由度提高
```

财务自由的可用路径：

| 路径 | 作用 | 当前策略 |
|------|------|----------|
| 工资提升 | 提高安全垫和本金积累速度 | 先从打开 LeetCode 破冰，后续恢复简历和面试 |
| 交易 | 长期复利能力 | 当前只学系统，不急实盘 |
| 副业 | 非工资现金流试验田 | 等睡眠和身体稳定后再选方向 |
| 稳健资产 | 未来被动收入底座 | 先控风险，不因亏损翻本 |

---

## 二、最快路径判断

当前最快路径不是 All in 交易，也不是立刻开副业。

当前最快路径是：

```
先恢复睡眠和身体 → 保住精力 → 启动工作提薪 → 低成本学习交易 → 等稳定后再试副业
```

原因：

- 当前疲劳 8/10，直接开新副业会增加崩盘风险。
- 交易系统未学完，实盘和翻本冲动风险高。
- 工资从 8.5k 提到 12k+ 的确定性高于短期交易盈利。
- 身体和睡眠是所有路径的放大器，先保住才有后续。

---

## 三、当前三阶段

### 阶段 1：恢复掌控感

目标：睡眠和身体先稳住。

最小动作：

- 22:30 手机放客厅。
- 23:00 躺下。
- 身体最低动作不断。

### 阶段 2：提高确定性现金流

目标：工作线从 0 启动，逐步提薪。

最小动作：

- 打开 LeetCode。
- 连续 7 天后，再恢复刷题和简历。

### 阶段 3：建立非工资现金流

目标：交易和副业都作为候选，不提前押注单一路径。

最小动作：

- 交易：每天 15min 看图或概念卡。
- 副业：等恢复档/正常档稳定后，再选择一个低成本试验。

---

## 四、系统原则

- 财务自由不绑定单一路径。
- 先提高确定性，再追求高弹性。
- 先让身体和睡眠稳定，再让收入结构变复杂。
- 任何赚钱路径都不能以长期健康崩盘为代价。
