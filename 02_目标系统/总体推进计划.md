## 三、目标联动机制

### 3.1 层级关系

```text
总体推进计划（本文件）
    ↓ 拆解
年度目标（2026年.md）
    ↓ 拆解
月度目标（2026-05.md 等）
    ↓ 拆解
周计划（第1周.md 等）
    ↓ 拆解
每日任务（2026-05-26.md 等）
```

### 3.1.1 执行层与归档层

| 区域 | 职责 |
|---|---|
| `02_目标系统/每日任务/` | 当前活跃窗口，仅保留正在执行的每日任务 |
| `02_目标系统/归档/每日任务/` | 历史每日任务归档，不参与日常启动 |

### 3.2 推进节奏

| 节奏 | 做什么 | 由谁 |
|---|---|---|
| **每日** | 执行当日任务，打勾 | 少爷执行，AI 记录 |
| **每周复盘**（周日/周一） | 用三问法检查周计划完成度 → 调整下周计划 | AI 发起 |
| **月度复盘**（月底） | 检查月度目标 → 调整下月目标 | AI 发起 |
| **阶段复盘**（阶段切换时） | 检查出口标准 → 启动新阶段 | AI 发起 |
| **系统维护**（按需） | 检查结构、归档、引用、镜像、一致性 | AI 发起或少爷触发 |

### 3.3 目标调整规则

| 场景 | 操作 |
|---|---|
| 月目标明显完不成（Week 2 就看出没戏） | 当周下调，不等到月底 |
| 某条线连续 3 天执行率 < 50% | 该线降级到最低版，直到恢复 |
| 意外收获（提前拿到 offer / 交易突然开窍） | 重新排优先级，但不盲目加量 |
| 外部冲击（生病 / 加班周 / 家庭事件） | 砍所有非必要线，只保身体 |
| 身体连续做顺后想猛加 | 不当天加量，统一放到周复盘决定，只允许微进阶 1 个变量 |

### 3.4 统一推进规则（基线 / 进阶 / 降级 / 产出 / 周预算）

#### 3.4.1 基线规则

| 主线 | 当前基线 | 说明 |
|---|---|---|
| 身体 | 完成当日最低训练版 | 先保不断线，再谈进阶 |
| 交易 | 完成一次低配学习动作 | 看课15min / 标记结构 / 语音笔记 三选一 |
| 现金流稳定线 | 完成一次 25min 低门槛推进 | 不要求时长堆叠，不要求日更 |

#### 3.4.2 进阶规则

| 主线 | 理论进阶方向 | 触发条件 |
|---|---|---|
| 身体 | 次数 / 时长微增 | 连续完成稳定 + 无明显不适 + 疲劳不过高 |
| 交易 | 从低配学习 → 更完整学习 / 模拟练习 | 学习频率稳定 + 有产出沉淀 |
| 现金流稳定线 | 从破冰 → 更高频推进 / 简历 / 小专题 | 每周能稳定推进，不再长期空转 |

#### 3.4.3 降级规则

| 触发情况 | 系统动作 |
|---|---|
| 疲劳 ≥ 8 | 所有主线降到最低版 |
| 连续 2 天睡眠崩 | 本周不允许任何主线进阶 |
| 某条线连续 3 次未完成 | 该线回到最低执行版 |
| 身体出现不适 / 肩前顶 | 身体线停止进阶，立刻回到安全基线 |

#### 3.4.4 产出规则

| 主线 | 最小产出 |
|---|---|
| 身体 | 完成训练 + 一句体感 |
| 交易 | 概念卡 / 结构标记 / 语音笔记 三选一 |
| 现金流稳定线 | 25min 推进 / 一条题型笔记 / 简历一小段 三选一 |

#### 3.4.5 周预算规则

| 项目 | 规则 |
|---|---|
| 每周最多进阶几条线 | 默认 1 条，最多 2 条 |
| 一条线最多进阶几个变量 | 1 个 |
| 系统维护是否占预算 | 默认不占，但不能抢主线注意力 |
| 周复盘的作用 | 只决定上调 / 维持 / 降级，不临时发明新大目标 |

#### 3.4.6 分线调速补充

| 主线 | 基线 | 允许进阶 | 降级触发 | 降级动作 | 停止条件 |
|---|---|---|---|---|---|
| 交易线 | 看课15min / 标记1个结构 / 语音笔记 三选一 | 连续 2 周有稳定频率，且每周至少有 2 次有效产出后，才允许从“学习”升级到“更完整学习或模拟练习” | 连续 3 次没做、睡眠连续 2 天崩、出现强烈翻本冲动 | 回到最低学习版，只保 15min 保底动作 | 连续熬夜、疲劳 ≥ 8、情绪明显受白酒仓位波动牵动时，本周停止进阶，只保最小学习动作 |
| 现金流稳定线 | 25min 低门槛推进 / 打开 LeetCode / 简历一小段 三选一 | 连续 2 周都能完成基线后，才允许从“破冰”升级到“每周 2-3 次推进”或“简历/题型小专题” | 连续 3 次没做、工作周压力明显升高、身体恢复被挤压 | 回到“打开就算开始”的破冰版，不强求完整产出 | 加班周、睡眠崩、身体线开始掉时，现金流稳定线暂停提速，只保最低接触 |

**补充约束：**

- 每周只允许 `1 条主进阶`，默认优先从最稳的那条线里选。
- 如果身体线当周不稳，交易线和现金流稳定线都不允许抢提速资格。
- 白酒相关仓位管理不算“交易线进阶”，也不算本周主进阶成果。

#### 3.4.7 决策日志与周复盘三问

##### 决策日志使用规则

| 场景 | 是否记录 | 记录位置 | 写法 |
|---|---|---|---|
| 切换主推线 | 是 | 当周周复盘 | `日期｜把什么提到主推｜原因｜观察到下次周复盘` |
| 某条线进阶 / 降级 | 是 | 当周周复盘 | `日期｜哪条线调档｜触发原因｜下周怎么验证` |
| 临时情绪波动下的小调整 | 否 | 不单独记录 | 直接体现在每日任务即可 |
| 单次没做完 / 单日状态差 | 否 | 不单独记录 | 只在每日任务或周复盘总结里提一句 |

##### 周复盘三问

| 问题 | 用途 |
|---|---|
| 这周最该维持的是哪条线，为什么？ | 防止把刚稳住的东西自己打乱 |
| 这周最该降级或止损的是哪条线，为什么？ | 及时止住空转、硬扛、假努力 |
| 下周唯一允许的主进阶是什么？ | 锁住节奏，避免三线同时乱加码 |

**规则：**

- 周复盘默认先回答三问，再决定下周计划。
- 如果三问答不出来，说明本周不该进阶，默认维持或降级。
- 决策日志只记“改变路线的决定”，不记流水账。

## 五、关键路径图

### 依赖关系

```text
身体稳定
  ├─→ 有精力学交易
  ├─→ 有精力刷题
  └─→ 好的睡眠

财务自由路线
  ├─→ 现金流稳定线：提高月现金流和安全垫
  ├─→ 交易学习：先模拟验证，再小资金实盘
  ├─→ 副业探索：优先选择低成本、可复用技能型副业
  └─→ 稳健资产：本金积累后作为底仓

白酒相关仓位
  └─→ 当前仅做风险监控 + 仓位管理，不计入主动推进路径

现金流稳定线（刷题 + 简历）
  ├─→ 面试
  │    └─→ 12k+ offer → 月攒 7k+ → 本金积累 → 财务自由路线加速
```

### 关键路径

```text
身体 → 精力 → 现金流稳定线/交易学习/副业探索 → 攒钱 + 现金流 → 自由
```

### 风险点

| 节点 | 风险 | 影响 | 备选方案 |
|---|---|---|---|
| 身体中断 | 高 | 其他两条线也会跟着断 | 中断协议启动，最低版恢复 |
| 睡前手机 | 高 | 熬夜 → 精力崩 → 次日全废 | 22:30 物理隔离手机 |
| 交易学不完 | 中 | 无法进入模拟盘 | 拉长学习周期，不赶进度 |
| 刷题不启动 | 中 | 无法拿 offer | 降到“打开就算完成” |

### 财务自由路线判断

| 路径 | 作用 | 当前策略 |
|---|---|---|
| 现金流稳定线 | 提高安全垫和本金积累速度 | 先从打开 LeetCode 破冰，后续恢复简历和面试 |
| 交易 | 长期复利能力 | 当前只学系统，不急实盘 |
| 副业 | 非工资现金流试验田 | 等睡眠和身体稳定后再选方向 |
| 稳健资产 | 未来被动收入底座 | 先控风险，不因亏损翻本 |

当前最快路径不是 All in 交易，也不是立刻开副业，而是：

```text
先恢复睡眠和身体 → 保住精力 → 低频恢复现金流稳定线 → 稳定学习交易 → 等状态稳后再扩展
```

原因：

- 当前疲劳高，直接开新副业会增加崩盘风险。
- 交易系统未学完，实盘和翻本冲动风险高。
- 现金流稳定线的确定性高于短期交易盈利，但也不能用高压日更方式推进。
- 身体和睡眠是所有路径的放大器，先保住才有后续。

### 当前资金约束提醒

| 项目 | 判断 |
|---|---|
| 白酒相关仓位 | 视为历史仓位管理，不是当前学习中的交易，也不是当前主路径 |
| 网格操作 | 作用是历史仓位的被动管理，不计入当前执行成果 |
| 系统判断原则 | 现金流稳定线、交易学习、身体恢复才是当前主动推进项；白酒仓位默认不纳入日常执行，只在维护/复盘时单独看 |

### 当前身体推进原则

| 项目 | 判断 |
|---|---|
| 当前阶段 | 身体线已经连续完成，不再是“重启问题”，而是“微进阶问题” |
| 进阶方式 | 先按周稳定，再一次只加一个变量 |
| 风险 | 最危险的不是练少，而是刚连续就猛加导致再次崩盘 |
