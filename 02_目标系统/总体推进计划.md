## 三、目标联动机制

### 3.1 层级关系

```
总体推进计划（本文件）
    ↓ 拆解
年度目标（2026年.md）
    ↓ 拆解
月度目标（2026-05.md 等）
    ↓ 拆解
周计划（第1周.md 等）
    ↓ 拆解
每日任务（2026-05-26.md 等）
```

### 3.2 推进节奏

| 节奏 | 做什么 | 由谁 |
|------|--------|------|
| **每日** | 执行当日任务，打勾 | 少爷执行，AI 记录 |
| **每周复盘**（周日/周一）| 检查周计划完成度 → 调整下周计划 | AI 发起 |
| **月度复盘**（月底）| 检查月度目标 → 调整下月目标 | AI 发起 |
| **阶段复盘**（阶段切换时）| 检查出口标准 → 启动新阶段 | AI 发起 |

### 3.3 目标调整规则

| 场景 | 操作 |
|------|------|
| 月目标明显完不成（Week 2 就看出没戏）| 当周下调，不等到月底 |
| 某条线连续 3 天执行率 < 50% | 该线降级到最低版，直到恢复 |
| 意外收获（提前拿到 offer / 交易突然开窍）| 重新排优先级，但不盲目加量 |
| 外部冲击（生病 / 加班周 / 家庭事件）| 砍所有非必要线，只保身体 |

## 五、关键路径图

### 依赖关系

```
身体稳定
  ├─→ 有精力学交易
  ├─→ 有精力刷题
  └─→ 好的睡眠

财务自由路径
  ├─→ 工作提薪：提高月现金流和安全垫
  ├─→ 交易学习：先模拟验证，再小资金实盘
  ├─→ 副业探索：优先选择低成本、可复用技能型副业
  └─→ 稳健资产：本金积累后作为底仓

工作刷题 + 简历
  ├─→ 面试
  │    └─→ 12k+ offer → 月攒 7k+ → 本金积累 → 财务自由路径加速
```

### 关键路径

```
身体 → 精力 → 工作提薪/交易学习/副业探索 → 攒钱 + 现金流 → 自由
```

### 风险点

| 节点 | 风险 | 影响 | 备选方案 |
|------|------|------|----------|
| 身体中断 | 高 | 其他两条线也会跟着断 | 中断协议启动，最低版恢复 |
| 睡前手机 | 高 | 熬夜→精力崩→次日全废 | 22:30 物理隔离手机 |
| 交易学不完 | 中 | 无法进入模拟盘 | 拉长学习周期，不赶进度 |
| 刷题不启动 | 中 | 无法拿 offer | 降到"打开就算完成" |
