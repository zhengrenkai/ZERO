# 归档说明

> 这里存放不再参与日常启动的历史执行资料。

---

## 归档规则

| 类型 | 存放位置 | 说明 |
|---|---|---|
| 历史每日任务 | `每日任务/YYYY-MM/` | 已失去当前入口价值，但需要保留追溯 |

## 当前规则

- `02_目标系统/每日任务/` 只保留当前活跃窗口
- 更早的每日任务进入 `02_目标系统/归档/每日任务/YYYY-MM/`
- 归档默认不参与 `启动`
- 只有在 `进入维护`、`维护系统`、`复盘` 时才按需读取
- 归档时未勾选项目默认视为未完成
- 若少爷事后补记完成或修正事实，必须同步：归档/当日每日任务 → 周计划 → 当前状态.json → 当前人生状态.md → 今日任务.md / 独立执行卡.md
- 事后补记只修正事实，不自动改变状态分数；状态分数需要少爷明确汇报或启动判断时再更新
