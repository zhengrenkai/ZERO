# 交易学习详细计划

> 交易之境 7 篇课程消化方案
> 已经看过一遍 → 现在是"用学习方法变成真会"

---

## 一、核心思路

课看完了但记不住 = 正常。你只做了"输入"，没做"输出"和"练习"。

**解决方案不是再看一遍，而是换一种方式学：**

```
上一次：看视频 → 听懂了 → 下一课（被动输入）
这一次：看视频 → 关掉 → 在图上找例子 → 给 AI 讲一遍（主动输出）
```

同一个视频，用主动输出的方式再看一遍，效果相当于看了 3 遍。

## 二、逐篇消化计划（8 周）

按照交易之境 7 篇的顺序，加上学习法则中的**80/20 法则**：
前三篇先消化，后四篇再跟。

### 阶段一：核心三篇（Week 1-4）

```
Week 1-2：第一篇 流动性原理 + 第二篇 市场结构
Week 3-4：第三篇 高概率POI区域 + 前三篇综合练习
```

### 阶段二：进阶四篇（Week 5-8）

```
Week 5-6：第四篇 价格传递 + 第五篇 背驰判断
Week 7-8：第六篇 多周期分析 + 第七篇 交易系统
```

## 三、每篇的消化流程（标准格式）

每篇课用同样的流程消化，不要跳步：

### Step 1：回看课程（30min）

不是从头看到尾。**逐段式回看**：

```
看 5 分钟 → 暂停 → 问自己：
"刚才这段讲了什么核心概念？"
→ 能回答 → 继续下一段
→ 不能回答 → 倒回去重看这 5 分钟
```

### Step 2：图上找例子（20min）

```
打开行情图（BTC 日线/4h/15min）
只练今天这篇的概念：
  流动性篇 → 找 3 个流动性区域
  市场结构篇 → 标出 HH/HL/LH/LL
  POI 篇 → 找 3 个 OB 或 FVG
```

**关键规则：必须标在图上。光靠脑子想不算找。**

### Step 3：费曼输出（10min）

```
对 AI 说：
"我用大白话讲一下 [今天学的概念]，
你帮我判断对不对，有没有漏掉什么。"

示例：
"流动性就是价格会去的地方，那里有止损单聚集。
前高前低是最常见的流动性区域。
我这样理解对吗？"
```

### Step 4：写一条概念笔记（5min）

每篇课消化完，产出一条笔记。格式：

```
## [概念名]

一句话理解：____
图上怎么看：____
常见误区：____
```

前三篇的概念笔记就是**Week 1 流动性图解、Week 2 订单块标注、Week 4 系统流程图**。

## 四、各篇要点（帮你抓重点）

### 第一篇：流动性原理

| 维度 | 内容 |
|------|------|
| 核心一句话 | 价格会去流动性聚集的地方扫止损 |
| 图上怎么看 | 标记前高前低 → 这些就是流动性区域 |
| 常见误区 | 以为所有高低点都是流动性 → 只有结构明显的地方才算 |
| 必须练到什么程度 | 打开任意图表，5 秒内标出流动性区域 |

### 第二篇：市场结构

| 维度 | 内容 |
|------|------|
| 核心一句话 | 上升趋势 = HH+HL，下降趋势 = LH+LL，结构破坏了就是趋势可能变了 |
| 图上怎么看 | 连趋势线 / 标 HH/HL/LH/LL / 找 MSS（市场结构转换）|
| 常见误区 | 只看小周期不看大周期 → 15min 的假突破在大周期可能只是回调 |
| 必须练到什么程度 | 打开任意图表，10 秒内判断当前趋势方向 |

### 第三篇：高概率 POI 区域

| 维度 | 内容 |
|------|------|
| 核心一句话 | POI（Points of Interest）= 价格可能会反应的地方，包括 OB（订单块）和 FVG（缺口）|
| 图上怎么看 | 找大阳线/大阴线的起点做 OB 标记 / 找连续 3 根不重叠的 K 线做 FVG 标记 |
| 常见误区 | OB 画得太宽 → OB 就是那 1-2 根 K 线，不是整段行情 |
| 必须练到什么程度 | 打开图表，30 秒内标出当前最有价值的 POI |

### 第四篇：价格传递

| 维度 | 内容 |
|------|------|
| 核心一句话 | 价格从一个流动性区域到另一个流动性区域的路径 |
| 图上怎么看 | 标记流动性池 → 看价格如何被"传递"到下一个池子 |
| 关键 | 理解"价格是如何流动的"，而不是只盯着入场点 |

### 第五篇：背驰判断

| 维度 | 内容 |
|------|------|
| 核心一句话 | 价格创新高但指标没创新高 = 背驰，趋势可能要反转 |
| 图上怎么看 | 价格和 RSI/MACD 对比，找背离点 |
| 关键 | 背驰是辅助信号，不是入场信号。等结构确认再入场 |

### 第六篇：多周期搭配分析

| 维度 | 内容 |
|------|------|
| 核心一句话 | 大周期定方向，小周期找入场点 |
| 具体操作 | 4h 看趋势方向 → 15min 找结构确认 → 1min 等入场 |
| 关键 | 大小周期方向一致才入场，不一致就等 |

### 第七篇：交易系统

| 维度 | 内容 |
|------|------|
| 核心一句话 | 把前 6 篇串起来，形成固定的交易流程 |
| 具体操作 | 每天按固定流程复盘 → 标记结构 → 找 POI → 等确认 → 入场 → 管理风险 |
| 关键 | 系统不是完美的，是**一致的**。一致性能帮你评估和学习 |

## 五、每日练习方案

### 工作日（1h/天）

```
20:10  今日交易练习（1h）
  ── 花 5 分钟：回顾昨天学的概念（间隔重复）
  ── 花 20 分钟：回看当前篇课程片段（逐段式）
  ── 花 20 分钟：图上找 3 个当前篇概念的实例
  ── 花 10 分钟：给 AI 讲一遍今天练的
  ── 花 5 分钟：写一条概念笔记
```

### 周末（3-4h/天，集中在上午）

```
上午 3-4h：
  ── 前 30 分钟：回顾本周学的所有概念
  ── 中间 2h：独立复盘当前行情（不用课程，自己分析）
  ── 最后 1h：给自己的分析打分 + 找课程对照修正
```

## 六、检验标准

### 每日检验

```
今天的目标概念，能不能：
① 用自己的话讲清楚（费曼测试）✅
② 在行情图上找到 3 个例子 ✅
③ 让 AI 听完觉得"理解正确" ✅
```

### 每篇检验（消化完一篇后）

```
不看课程，独立做：
① 写出这篇的一页笔记（5 个要点）
② 在 3 张不同图表上应用这篇的概念
③ 给 AI 讲一遍，AI 说 OK
```

### 阶段检验（四篇消化完后）

```
不看任何资料：
① 打开当前 BTC 行情图，做一次完整的复盘分析
  ── 标流动性区域
  ── 标当前趋势结构
  ── 标 POI 区域
  ── 给出明日预期
② 让 AI 评价分析质量
```

## 七、与学习法则的结合

| 法则 | 在哪用 | 怎么用 |
|------|--------|--------|
| 费曼 | 每天学完 | 给 AI 讲一遍今天的概念 |
| 主动回忆 | 复习时 | 关掉课程，在图上自己标 |
| 间隔重复 | 每天开始 | 花 5 分钟回顾前几天学的 |
| 刻意练习 | 每篇课程 | 今天只练这一个概念，不贪多 |
| 80/20 | 学顺序 | 先消化前三篇核心，再跟后面 |
| AI 辅助 | 全过程 | 让 AI 判对错、出检验题 |
