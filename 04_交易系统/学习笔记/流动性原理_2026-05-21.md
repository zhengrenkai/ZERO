## 流动性原理

**一句话理解：** 主力利用流动性设置陷阱与推动价格的工具。流动性 = 有潜在止损单和挂单的位置，主力通过清扫这些流动性来吸筹或派发。

**图上怎么看：**

| 类型 | 位置 | 主力意图 |
|------|------|----------|
| 卖方流动性 | 结构低点（止损单 + 做空挂单） | 扫掉 → 获取低成本筹码 |
| 买方流动性 | 结构高点（突破单） + OB/FVG（低成本筹码） | 扫掉 → 用更少成本推价格到目标位 → 派发获利 |

**PO3 原理（积累·操纵·派发）：**

```
积累 → 吸筹
操纵 → 砸穿结构低点扫卖方流动性（假突破），拿到筹码
派发 → 拉到买方流动性区域出货获利
```

**实战滤波原则：**
1. 先定大周期方向（4h/日线）
2. 只找与当前方向**相反**的流动性清扫
   - 上升趋势 → 关注结构低点被扫（吸筹码号）
   - 下降趋势 → 关注结构高点被扫（派发信号）
3. 其他流动性清扫 → 忽略

**常见误区：** 觉得哪都是流动性清扫。正确做法是先定方向，只关注与方向相反的清扫。
