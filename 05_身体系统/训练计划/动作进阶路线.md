# 动作进阶路线

> 你的街健成长路径。Phase 1→2→3，每阶段升级一次动作难度。
> 不跳阶，不急进，以肩关节安全为底线。

---

## 总览

```
Phase 1（当前）：体态修复 + 基础控制
       ↓ 连续执行 14 天 + 动作熟练
Phase 2：力量进阶 + 技能学习
       ↓ 稳定执行 60 天 + 基础力量达标
Phase 3：街健动作组合
       ↓ 长期保持
街健美学体型 ✅
```

---

## 各动作进阶路线图

| Phase | 动作 | 目标 | 通往 |

| Phase | 动作 | 目标 | 通往 |

### 🔹 靠墙倒立 → 倒立控制 → 自由倒立

## 进阶条件（什么时候升 Phase）

### Phase 1 → Phase 2 条件

| 条件 | 具体标准 |
|------|----------|
| 连续执行 | 14 天无中断（允许最低执行版）|
| 墙天使 | 手臂贴墙幅度明显改善 |
| 肩胛俯卧撑 | 能清晰感受前锯肌发力 |
| 引体 | 3×4 轻松完成，不耸肩 |
| 靠墙倒立 | 3×20秒 稳定，核心不塌 |

> 不需要全部完美，主要看「是否建立了持续执行的节奏」

### Phase 2 → Phase 3 条件

| 条件 | 具体标准 |
|------|----------|
| 连续执行 | 60 天稳定 |
| 引体 | 4×8 以上 |
| 靠墙倒立 | 3×45秒 稳定，单腿离墙能保持 3 秒 |
| 体态 | 脖子前倾明显改善，站立自然挺直 |

---

## 重要提醒

- **Phase 1 不要想着 Phase 3 的动作**，你现在的任务是让身体适应每天动一动
- 每个 Phase 至少执行 4~8 周，不急于进阶
- 肩前顶、肩峰疼痛 → 退回上一阶或做恢复日
- 进阶不是终点，稳定执行才是
