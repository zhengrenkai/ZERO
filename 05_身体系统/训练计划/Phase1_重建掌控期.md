# Phase 1: 重建掌控期 - 训练计划

## 核心原则
- 每天15-20分钟
- 4个核心动作
- 不力竭、不加量
- 熬夜或状态差 → 最低执行版
- **饭后休息30~40分钟再练**，刚吃完不练引体和倒立

## 每日动作

### 1. 墙天使 2×10
修复脖子前倾+圆肩
标准：腰别离墙，下巴微收，手臂尽量贴墙

### 2. 肩胛俯卧撑 2×10
肩关节保护，前锯肌激活
标准：手臂不弯，只动肩胛

### 3. 引体向上 3×4
保留力量，非力竭
标准：不耸肩，下放控制2秒

### 4. 靠墙倒立 3×20秒
神经控制，肩胛稳定
标准：核心收紧，不塌腰

## 最低执行版（状态差时）
- 肩胛俯卧撑 2×10
- 引体 3×3
- 靠墙倒立 3×20秒
总时间：10分钟

## 期限
先连续执行14天不断。
完成后评估进入 Phase 2。
