# 每日生活流程

> Phase 1 版本。不追求一步到位，先跑通基础流程。

---

## 晨间流程（~20 分钟）

```
起床 ─┬─ 喝水 600ml
      ├─ 补剂（详见下方「补剂周期表」）
      ├─ 护肤：清水洗脸 → 喷雾 → 面霜
      ├─ 护手霜
      └─ 早餐
```

## 工作日白天

```
 9:00-12:00  上班 💼
12:00-13:30  午休 ─ 午饭 + 喝水 600ml + 补水喷雾（专心吃饭休息）
13:30-18:00  上班 💼
```

## 晚间流程（核心）

```
18:00  下班

18:00-19:00  通勤 + 晚饭 🍚
             💊 晚饭补剂（鱼油 + D3K2 + 叶黄素）

19:00  到家 ── 休息 30~40 分钟 📱
────────────────────────────────
19:40  锻炼开始 💪
  □ 猫牛式 ×5（开胸椎）
  □ 墙天使 2×10
  □ 肩胛俯卧撑 2×10
  □ 引体 3×4
  □ 靠墙倒立 3×20秒
────────────────────────────────
19:55  锻炼结束 ✅

  洗衣服 👕（丢洗衣机）
  洗澡 🚿
  护肤：洗面奶 → 喷雾 → 面霜
  护手霜

  自由时间 🎮

20:10  交易产出 💹
  □ 行情结构笔记（标 1 个结构点 + 一句话）
  □ 或 概念卡片（写一句理解 + 找 1 个例子）
────────────────────────────────
21:10~23:30  自由时间 🎮

23:30  睡前仪式
  □ 放下手机
  □ 关大灯，开小夜灯
  □ 蒸汽眼罩

00:00~00:30  躺下 🛌
```

## 当前默认规则

| 规则 | 说明 |
|------|------|
| 训练 | 每天 猫牛式+4 个核心动作，15~20 分钟 |
| 加量 | **禁止**，状态好也不加 |
| 状态差 | 切换最低执行版（砍到 3 项，10 分钟）|
| 饭后 | 休息 30~40 分钟再练 |
| 肩痛 | 立刻停，不硬顶 |
| 睡眠 | 目标 12:30 躺下，不追求一步到位 |
| 中断 | 断了直接继续，不责备 |
| 午休 | 专心吃饭休息，刷题不强求 |
| 交易产出 | 每天出一个就行（行情结构笔记 OR 概念卡片），不是两个都必须做 |
| 刷题 | 晚上交易产出做完后当换脑子，不硬性规定 |
| 三线战报 | 每次对话更新三线状态 + 疲劳自评 |
| 砍线 | 疲劳 ≥ 8 砍工作线，只保身体+交易 |
