## 三线战报子格式

```
身体：✅ / ⬜ / ❌
交易：✅ / ⬜ / ❌
工作：✅ / ⬜ / ❌
疲劳：_/10
```

### 符号解释

| 符号 | 含义 |
|------|------|
| ✅ | 完成（身体：4 个动作+体感 / 交易：1 个产出物 / 现金流稳定线：本周攒够题型笔记） |
| ⬜ | 做了但产出不够（身体：只做了最低版 / 交易：看了行情但没产出 / 现金流稳定线：刷了但没攒笔记） |
| ❌ | 没做 |

### 现金流稳定线标准

现金流稳定线按周统计：

| 状态 | 条件 |
|------|------|
| ✅ 达标 | 本周攒了 ≥ 4 条题型笔记 |
| ⬜ 低标 | 本周攒了 1-3 条 |
| ❌ 没做 | 本周 0 条 |

### 示例

```
身体：✅
交易：⬜（看了行情但没写出结构点）
现金流稳定线：✅（这周攒了 5 条题型笔记）
疲劳：7/10
```

```
身体：❌
交易：❌
现金流稳定线：⬜
疲劳：9/10
→ 触发砍线机制：砍现金流稳定线，明天从最低版开始
```
