# 工作流程

## 启动流程（每次新对话）

1. 读取 00_系统核心/人生逆袭实验.md — 实验框架
2. 读取 00_系统核心/AI系统自身概要.md — 系统身份
3. 读取 01_人生核心档案/ 全部 6 个 JSON — 记忆恢复
4. 读取 08_仪表盘/ — 当前状态
5. 读取 02_目标系统/总体推进计划.md — 当前推进阶段和下一步行动
6. 读取 09_系统建设/优化建议池.md — 待办建设任务
7. 分析状态，检查未处理观察
8. 执行对话开场：输出简报 + 推进位置，引导少爷汇报

## 每日对话流程

### 1. 开场
- 输出状态简报（含身体/交易/工作三条线）
- 引导少爷汇报昨日执行

### 2. 少爷汇报后
- 确认执行结果
- 更新 5 维状态（精力/动量/纪律/疲劳/压力）
- 如有风险预警
- 给出今日策略

### 3. 对话结束时
- 写入 03_执行记录/每日执行日志/
- 更新 01_人生核心档案/当前状态.json
- 如有新观察 → 追加 AI观察记录.json

### 4. 推送更新（结束前）
- git status 确认改动范围
- git commit + push，备注按 conventional commit 格式：
  - `<type>: <每次改了什么>`
  - 多行：第一行标题，空行后写详细内容
  - 不推送 .idea/ 目录

## 每周流程（周末执行）

- 读取本周每日执行日志（含身体/交易/工作）
- 分析三条线的执行趋势
- 更新行为模式
- 确定下周优化方向
- 写入 06_复盘系统/每周复盘/

## 每月流程（月末执行）

- 分析月趋势（三条线分别评估）
- 评估阶段进展
- 调整季度目标
- 更新长期观察
- 写入 06_复盘系统/每月复盘/
