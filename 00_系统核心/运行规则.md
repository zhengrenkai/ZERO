# 运行规则

## 目录结构
Jarvis-人生系统/
├── 00_系统核心/           # AI 行为规则
├── 01_人生核心档案/       # 持久化记忆（6个核心 JSON）
├── 02_目标系统/           # 各阶段目标与计划
├── 03_执行记录/           # 每日执行日志
├── 04_交易系统/           # 交易相关
├── 05_身体系统/           # 身体训练相关
├── 06_复盘系统/           # 复盘记录
├── 07_AI观察与动态调整/   # AI 长期观察
└── 08_仪表盘/             # 当前状态看板

## 核心文件职责

### 01_人生核心档案/人格与方向.json
最核心文件。保存：
- 你是谁
- 长期方向
- 核心价值观
- 性格特征
- 人生原则
永不频繁修改，仅在重大认知升级时更新。

### 01_人生核心档案/长期目标.json
保存：
- 长期目标（2个）
- 当前阶段名称与描述
- 季度目标（active / pending）
- 目标优先级

### 01_人生核心档案/行为模式.json
AI 最重要的智能来源。保存：
- 拖延规律
- 崩溃规律
- 高效规律
- 情绪规律
每次发现新规律时追加。

### 01_人生核心档案/当前状态.json
实时状态。保存：
- 精力（1-10）
- 动量/Momentum（1-10）
- 纪律/Discipline（1-10）
- 疲劳（1-10）
- 压力（1-10）
- 当前阶段
每次启动时读取，每次对话结束时更新。

### 01_人生核心档案/AI观察记录.json
AI 的观察记录。保存：
- 最近的问题
- 风险
- 偏离
- 趋势
- 策略建议

### 01_人生核心档案/环境信息.json
外部环境。保存：
- 工作环境
- 可支配时间
- 收入状况
- 资源与工具

## 复盘节奏
- 每日：执行记录 → 03_执行记录/每日执行日志/
- 每周：趋势分析 + 下周优化方向
- 每月：系统成长速度 + 自律稳定度 + 长期风险

## 版本控制
- 核心文件变更应记录在 AI观察记录.json 中
- 重大变更（如阶段切换）需在 当前状态.json 中更新 phase
