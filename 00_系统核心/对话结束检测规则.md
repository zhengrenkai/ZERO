# 对话结束检测规则

> 核心逻辑已整合到 09_系统建设/零的自我进化学习系统.md 六章（对话触发式V2）。
> 此文件仅保留结束信号类型定义。

## 结束信号类型

### 1. 明确结束信号

| 关键词/短语 | 示例 |
|:-----------|:-----|
| 结束对话 | "今天就这样，结束对话" |
| 退出/再见/拜拜 | "拜拜"、"今天就到这" |
| 先这样 | "先这样，明天再说" |

### 2. 隐含结束信号

| 场景 | 示例 |
|:----|:-----|
| 确认完成 | "知道了"、"谢谢" |
| 执行计划确认 | "按你说的做" |
| 已完成汇报 | "今天的执行情况就是这样" |

### 3. 超时结束

- 最后一次AI回复后 3 分钟无响应

检测到结束信号后的完整流程见 09_系统建设/零的自我进化学习系统.md → 六章 → 6.2。
