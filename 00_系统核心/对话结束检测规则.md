# 对话结束检测规则

> 定义零如何判断对话何时结束。

---

## 结束信号类型

### 1. 明确结束信号

| 关键词/短语 | 示例 |
|:-----------|:-----|
| 结束对话 | "今天就这样，结束对话" |
| 退出/再见/拜拜 | "拜拜"、"今天就到这" |
| 先这样 | "先这样，明天再说" |

### 2. 隐含结束信号

| 场景 | 示例 |
|:----|:-----|
| 确认完成 | "知道了"、"谢谢" |
| 执行计划确认 | "按你说的做" |
| 已完成汇报 | "今天的执行情况就是这样" |

### 3. 超时结束

- 最后一次AI回复后 3 分钟无响应

---

## 结束流程

检测到结束信号后，执行 00_系统核心/工作流程.md → 对话结束时章节：

1. **状态保存** — 写入每日执行日志 + 更新当前状态
2. **自我进化扫描** — 自动扫描观察并生成优化建议
3. **优化效果回看** — 检查已实施建议的效果
4. **推送更新** — git commit + push
