## 七、自我进化循环流程

### 7.1 每周进化循环

```
周一：学习日
  → 读取上周观察和执行数据
  → 识别知识缺口
  → 确定本周学习目标
  → 学习并记录

周三：推演讲习日
  → 运行沙盒推演
  → 识别风险和优化点
  → 生成优化建议

周五：执行检查日
  → 检查本周执行情况
  → 记录观察
  → 评估优化效果

周日：进化整合日
  → 汇总本周学习/推演/执行
  → 更新系统规则和策略
  → 确定下周计划
```

### 7.2 触发式进化

| 触发条件 | 进化动作 |
|----------|----------|
| 连续3次相同失败模式 | 自动添加新规则到 AI行为规则.md |
| 优化建议实施后效果不佳 | 重新推演并调整方案 |
| 系统健康度 < 40 | 自动进入"修复模式"，暂停非核心任务 |
| 观察到新的成功模式 | 写入行为模式.json |

## 十一、核心规则

1. **持续学习**：每周至少学习1个新主题
2. **实践导向**：学习的知识必须能应用到系统中
3. **数据驱动**：进化决策基于观察数据而非直觉
4. **反馈闭环**：实施优化后必须评估效果
5. **勇于试错**：允许尝试新方法，失败了就调整
