## 二、18个月路线图

### Month 1-3：筑基期

| 现金流稳定线 | 交易线 | 身体线 |
|--------|--------|--------|
| 刷题 + 更新简历 + 面12k+ | 学完SMC系统 + 每天复盘行情 | Phase 1 每日4动作 |
| 目标：3个月内拿到offer | 每天看盘30分钟，不开实盘 | 连续执行60天 |

**Month 1 具体任务**：
- 现金流稳定线：碎片化刷题，每周累计 2h 就算达标
- 交易：按课程7篇顺序推进，每2周一主题
  - Week 1-2：第一篇 流动性原理 → 第二篇 市场结构
  - Week 3-4：第三篇 高概率POI区域 → 前三篇综合练习
- 记录：每天 10min 行情笔记（只记录今天看到的 1-2 个结构，不用多）

**Month 2 具体任务**：
- 现金流稳定线：开始投简历 + 面试积累经验
- 交易：模拟盘执行SMC系统，记录每笔交易理由
- 里程碑：能识别出SMC的流动性区域/订单块/市场结构

**Month 3 具体任务**：
- 现金流稳定线：目标是拿到offer或至少到终面
- 交易：微型实盘（≤1000），目标不是赚钱，是不亏钱
- 里程碑：实盘能执行系统而不被情绪带偏

## 三、前3个月每日/每周节奏

### 工作日（Month 1-3）

| 时间 | 内容 |
|------|------|
| 9:00-12:00 | 上班 |
| 午休 | 吃饭 + 休息（刷题不强求，改到晚上碎片时间） |
| 13:30-18:00 | 上班 |
| 晚饭后 | 训练 15-20 分钟 |
| 晚上 1-2h | **交易学习为主**（复盘行情/学SMC），状态好多出来的时间刷题 |

> 前3个月以身体+交易为优先。现金流稳定线不做硬性要求，碎片化刷题，每周累计 2h 就算达标。如果实在太累，优先保身体+交易。

### 周末（Month 1-3）

| 时间 | 内容 |
|------|------|
| 上午 3-4h | 交易：回测/系统学习（精力最好的时段给交易） |
| 下午 | 自由时间 + 训练 |
| 晚上 1-2h | 交易复盘（30min）+ 碎片刷题（30min）+ 改简历（30min） |

## 四、风控红线

1. **实盘单笔亏损不超过本金2%**（按实际交易本金单独计算，不用总资产替代）
2. **单月亏损超过15%暂停实盘**，退回模拟盘
3. **前3个月不开实盘**（第1个月学系统，第2个月模拟盘，第3个月微型盘）
4. **如果第6个月还没摸到盈利门道** → 不慌，交易本来就是长跑。保持现金流稳定线，继续磨
5. **绝对禁止在情绪差/熬夜后/大亏后"捞回"**

```
## 专题：Binary Search
模板：
  while (left <= right):
    mid = left + (right - left) // 2
    if nums[mid] == target → return
    elif nums[mid] < target → left = mid + 1
    else → right = mid - 1

适用条件：有序数组 / 找分界点
例题：704. Binary Search
```
